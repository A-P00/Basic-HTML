Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It was developed by Tim Berners-Lee in 1991.

Note: HTML describes the structure of web pages using markup elements that browsers interpret to display content.

HTML Versions

Version Year Significance
HTML 1.0 1993 The first public version
HTML 2.0 1995 Standardized by IETF
HTML 3.2 1997 W3C recommendation
HTML 4.01 1999 Strict, Transitional, Frameset
XHTML 1.0 2000 XML-based HTML
HTML5 2014 Current standard with new semantic elements

Basic HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Key Elements:

  • <!DOCTYPE html> - Declares HTML5 document type
  • <html> - Root element
  • <head> - Contains meta information
  • <title> - Specifies page title
  • <body> - Contains visible content

HTML Elements

1. Container Elements

Elements with both opening and closing tags:

  • <p>Paragraph</p>
  • <div>Division</div>
  • <span>Inline span</span>

2. Void Elements

Self-closing elements with no content:

  • <br> - Line break
  • <img> - Image
  • <input> - Form input

Headings

HTML provides six levels of headings:

<h1>Heading Level 1</h1> <h2>Heading Level 2</h2> <h3>Heading Level 3</h3> <h4>Heading Level 4</h4> <h5>Heading Level 5</h5> <h6>Heading Level 6</h6>

Best Practice: Use headings hierarchically (h1 → h2 → h3) for document structure and SEO.

HTML Tags and Their Explanation

  • <!DOCTYPE html> - Declares the document type and HTML version
  • <meta charset="utf-8"> - Specifies the character encoding for the document
  • <html>...</html> - The root element of an HTML page
  • <head>...</head> - Contains meta-information about the document
  • <title>...</title> - Specifies a title for the document
  • <body>...</body> - Contains the visible page content

Text Formatting

Tag Description Example
<b> Bold text Bold
<i> Italic text Italic
<u> Underlined text Underlined
<em> Emphasized text Emphasized
<mark> Highlighted text Highlighted
<small> Smaller text Small
<sub> Subscript H2O
<sup> Superscript x2

Note: While <b> and <i> still work, HTML5 recommends using <strong> and <em> for semantic meaning.

Basic Formatting Tags

Tag Description
<br> Line break (empty/void element)
<p>...</p> Paragraph (use CSS for alignment instead of align attribute)
<hr> Horizontal rule (empty/void element)
<!-- --> HTML comment (not displayed in browser)
size Height in pixels (deprecated in HTML5 - use CSS instead)
color Color of the line (deprecated in HTML5 - use CSS instead)
width Width in pixels or percentage (deprecated in HTML5 - use CSS instead)
noshade Specifies a solid line (deprecated in HTML5 - use CSS instead)

HTML5 Semantic Elements

Element Description
<header> Introductory content
<nav> Navigation links
<main> Main content
<article> Independent content
<section> Thematic grouping
<aside> Sidebar content
<footer> Footer content

Tip: Semantic elements improve accessibility and SEO by clearly defining content structure.

Tables

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Table Elements:

  • <table> - Defines the table
  • <tr> - Table row
  • <th> - Table header
  • <td> - Table data cell
  • <caption> - Table caption
